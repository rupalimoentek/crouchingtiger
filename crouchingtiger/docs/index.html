Installation Get Forza up and running

The easiest way to get Forza up and running is with the prebuilt folder we've provided.

First you'll need an HTTP server. If you're on Windows, check out XAMPP.

Once you've installed an Apache server, just paste the prebuilt in the htdocs folder. Now visit http://localhost/prebuilt/ and Forza should be up and running!

Build On the build flow of Forza

Forza is built with the modern build flow that incorporates Bower and Grunt to automate the build process.

NOTE If you wish to skip building the project manually just use the prebuilt folder instead that doesn't require this step.

NOTE we assume you already have Node/NPM installed on your system.

Installing Grunt

First install Grunt command line interface

npm install -g grunt-cli

The necessary package.json and Gruntfile.js are already there in the folder.

Installing Grunt Dependencies

Once Grunt is installed, cd to the project folder and run:

npm install

This will install the necessary dependencies for running Grunt tasks.

Installing Bower

Run the following command to install Bower globally:

npm install -g bower

Now you have Bower installed!

Installing Bower Dependencies

Now you should install the Bower packages required by Forza. To do this:

bower install

It'll install all the necessary packages for Forza in the app/bower_components folder.

Building the project

When you want to build the project for production, simply run

grunt

You may have to use sudo if you lack permissions.

Markup Markup customization and highlights

The theme is fluid layout with 12 columns. The left sidebar is removed from the grid and is fixed. The right sidebar appears on top of site and is also not a part of the grid. All content should go inside #page-content > #wrap.

Partial Views

Standard naming convention

Wildcard routes look for a .html file with the name matching the URL segment after the #/.

So, when you hit the URL #/layout-fixed, Forza will make a request to the server for layout-fixed.html partial view file located in app/views/.

Fixed/Boxed Layout

You can easily make a fixed/boxed layout by adding the fixed-layout class to the body in the above code. You can change the width of the boxed layout and the background from the variables.less file.

In JavaScript, you can use the provided $global service to turn on boxed layout with $global.set('layoutBoxed', true).

Fullscreen Layout

The body needs a focusedform class to turn on fullscreen mode.

To do this, turn on fullscreen with $global.set('fullscreen', true).

Horizontal Layout

The body needs a layout-horizontal class to use the horizontal layout.

To turn on horizontal layout programmatically, use $global.set('layoutHorizontal', true).

Navigation

Left sidebar navigation controls

Forza has a unique and completely data-driven navigation system keeping true to the Angular methodology. Therefore, it's split between view and model parts.

First, the view is set up to recursively render the menu items using the templates/nav_renderer.html template snippet.

  <ul ng-controller="NavigationController" id="sidebar">
      <li ng-repeat="item in menu"
          ng-class="{ hasChild: (item.children!==undefined),
                        active: item.selected,
                          open: (item.children!==undefined) && item.selected }"
          ng-include="'templates/nav_renderer.html'"
        ></li>
  </ul>

Note that the behavior is driven by the NavigationController controller. This is defined in the app/scripts/controllers/Navigation.js file.

Here's how menu items are built in JS:

$scope.menu = [
    {
        label: 'Dashboard',
        iconClasses: 'fa fa-home',
        url: '#/',
        html: '<span class="badge badge-indigo">4</span>',
    },
    {
        label: 'Layout Options',
        iconClasses: 'fa fa-cog',
        children: [
            {
                label: 'Grids',
                url: '#/layout-grid'
            },
            {
                label: 'Horizontal Navigation',
                url: '#/layout-horizontal'
            },
            {
                label: 'Horizontal Navigation 2',
                url: '#/layout-horizontal2'
            },
            {
                label: 'Fixed Boxed Layout',
                url: '#/layout-fixed'
            }
        ]
    }

...

]

The $global service

Forza comes with an Angular service named $global which you can inject into any controller that wishes to change global template properties.

This hopes provide a singular entrypoint for all UI properties that may need to be changed without relying on nesting scopes.

For example, you use the following line to change into fullscreen layout and restore previous layout on scope destroy:

    $global.set('fullscreen', true);

    $scope.$on('$destroy', function () {
      $global.set('fullscreen', false);
    });

Any other controllers listening for changes to service properties will be notified of this change.

Fonts Custom Fonts

Webfonts

Spice up your site with webfonts!

Forza uses Source Sans Pro and Source Sans Pro Light webfonts. Include the following script at the very start the page right after the opening head tag of your page (learn why) to load the fonts asnychronously. You may change the default fonts from the variables.less file

<script type="text/javascript">
  WebFontConfig = {
    google: { families: [ 'Source+Sans+Pro:300,400,600:latin' ] }
  };
  (function() {
    var wf = document.createElement('script');
    wf.src = ('https:' == document.location.protocol ? 'https' : 'http') +
      '://ajax.googleapis.com/ajax/libs/webfont/1/webfont.js';
    wf.type = 'text/javascript';
    wf.async = 'true';
    var s = document.getElementsByTagName('script')[0];
    s.parentNode.insertBefore(wf, s);
  })(); </script>

Or alternatively

<link href='http://fonts.googleapis.com/css?family=Source+Sans+Pro:300,400,600' rel='stylesheet' type='text/css'>

Icon Font

Loation of the icon fonts

Forza uses FontAwesome icon library as default.

The library is included as a less file in the main styles.less

If you want to separately include the FontAwesome library, you must include the CSS file from 

<link href="assets/fonts/font-awesome/css/font-awesome.min.css" rel="stylesheet">

If you wish to, however, you can include the font set from the following CDN

<link href="//netdna.bootstrapcdn.com/font-awesome/4.0.1/css/font-awesome.min.css" rel="stylesheet">

To include the glyphicon iconset, you must similarly include the CSS file from the link below. Note that Font-Awesome is used primarily in Forza and you might need to modify Forza significantly to move to this iconset. However, you can use Glyphicon and Font-Awesome side-by-side

<link href="assets/fonts/glyphicons/css/glyphicons.css" rel="stylesheet">

Compability Fixes

Some plugins listed requires special modifications to work with Bootstrap 3, and thus, the theme.

Form Wizard / Stepy

Add the following line of code after you call the .stepy() plugin. This ensures that the back and next buttons are properly indented to the panel-footer. jQuery.steppy.js has been lightly modified with classnames used in Bootstrap 3

$('.stepy-navigator').wrapInner('<div class="pull-right"></div>');

Form Validation / jQuery Validation

Add these properties to the jQuery Validation plugin to make it work with Forza. Note that this is not the plugin demonstrated in the Form Validation demo of the site but just with the wizard. The jQuery Validator support is built in to the Wizard

$('#form').validate({
    errorClass: "help-block",
    validClass: "help-block",
    highlight: function(element, errorClass,validClass) {
       $(element).closest('.form-group').addClass("has-error");
    },
    unhighlight: function(element, errorClass,validClass) {
        $(element).closest('.form-group').removeClass("has-error");
    }
});

Form Validation / Parsley

Add these properties whenever you use Parsley

//Parsley Form Validation
//While the JS is not usually required in Parsley, we will be modifying
//the default classes so it plays well with Bootstrap
$('#vform).parsley({
    successClass: 'has-success',
    errorClass: 'has-error',
    errors: {
        classHandler: function(el) {
            return $(el).closest('.form-group');
        },
        errorsWrapper: '<ul class=\"help-block list-unstyled\"></ul>',
        errorElem: '<li></li>'
    }
});

Theme Tips Make the most out of Forza

Always show navigation arrows instead of when just hovering

Find and change the first instance of #sidebar:hover li.hasChild>a:before, #sidebar:hover li.hasChild.open>a:before in the file layout.less and remove all :hover references

Leverage LESS Mixins

Create as many different colors schemes of panels or buttons as you want!

Simply change and add the variables in panels.less file:

.panel-color { 
  .panel-variated(@heading-text-color, @heading-bg-color, @heading-border);
}
For Buttons in buttons.less: 
.btn-color {
  .btn-pseudo-states(@btn-default-color, @btn-default-bg, @btn-default-border);
}
For Alternative Buttons: 
.btn-color-alt {
  .btn-alt-states(@border);
}
For alerts in alerts.less: 
.alert-color {
  .alert-variant(@background; @border; @text-color);
}
For labels in labels.less: 
.label-color {
  .label-variant(@color);
}
For progressbars in progress-bars.less: 
.progress-color {
  .progress-bar-variant(@color);
}

8 and below

For supporting IE8 and below, please include the following in your header

<!--[if lt IE 9]>
  <link rel="stylesheet" href="assets/css/ie8.css">
  <script type="text/javascript" src="http://html5shim.googlecode.com/svn/trunk/html5.js"></script>
  <script type="text/javascript" src="http://cdnjs.cloudflare.com/ajax/libs/respond.js/1.1.0/respond.min.js"></script>
  <script type="text/javascript" src="assets/plugins/charts-flot/excanvas.min.js"></script>
<![endif]-->

Respond.js, which makes Bootstrap IE8 compatible, is inherently incompatible with LESS.js because of how they work (learn more). If you want your site to function properly in IE8 we strongly recommend that you compile your LESS files into a CSS file - it also improves performance!

The included assets/js/placeholdr.js adds drop-in HTML5 placeholder support for input boxes. However, it must be executed after jQuery

FAQ

What is LESS and how do I use it?

LESS is a CSS Pre-Processor. In simple terms, LESS allows you to write CSS in a smarter way by combining functions, mixins, operations and more. This means you write more concise style information and can reuse things like colors and styles more easily.

Bootstrap was built using LESS, which is why we also use LESS to build Forza. While a full tutorial is out of the scope of the documentation, there are many tutorials of LESS available on the internet.

More to the point, all the .less files are in assets/less directory of the theme. Browsers cannot read less files as is, which is why we need to compile it into css. There are many applications which can serve this purpse (WinLESS, SimpLESS) or you can also use javascript to read less directly (slower, we recommend compiling into css first in production environment).
